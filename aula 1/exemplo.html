JS DOM

Métodos para Acessar Elementos no JavaScript

Escolha o método

document.getElementById(id)

Busca no documento um elemento que tenha o atributo id igual ao valor informado. Retorna exatamente um elemento, ou null se não encontrar.

Retorna: Element | null 
// Seleciona o elemento com id="caixa-info"
const elemento = document.getElementById('caixa-info');

console.log(elemento.textContent);
elemento.style.background = '#fff9c4';
